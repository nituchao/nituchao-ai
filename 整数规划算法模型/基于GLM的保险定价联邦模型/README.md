## 文件说明

| 文件名 | 说明 |
|:---|---|
|data| 保险数据集 |

## Tweedie/Poisson/Gamma回归样本

This example illustrates the use of Poisson, Gamma and Tweedie regression on the [French Motor Third-Party Liability Claims dataset](https://www.openml.org/d/41214)
